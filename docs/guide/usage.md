# 日常使用

## 查看用量与账单

在气泡水平台侧边栏：

- 「**用量**」查看每日请求次数、token 消耗与成本明细
- 「**我的订单**」查看历史订单与支付状态
- 「**购买订阅**」在线充值，支持支付宝、微信等

## 管理 API Key

- 「**API 密钥**」页面可创建、删除、重命名 API Key
- 每个 Key 可独立设置配额与有效期
- 密钥仅在创建时完整展示一次，请妥善保存

## 切换模型

在 CC Switch 中：

1. 打开「**气泡水**」提供商设置
2. 在模型列表中选择所需模型（如 `gpt-4o`、`claude-3.5-sonnet` 等）
3. 保存后重启 ChatGPT 生效

## 多账号管理

如需使用多个 API Key：

1. 在气泡水平台创建多个 Key
2. 在 CC Switch 中添加多个「气泡水」提供商
3. 通过 CC Switch 快速切换不同 Key

::: tip
不同 Key 的用量独立计费，便于团队或项目隔离。
:::
